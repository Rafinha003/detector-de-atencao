# Attention Detector

Sistema de visão computacional em **tempo real** para detecção de atenção do usuário, utilizando **webcam**, **OpenCV** e **MediaPipe**.

## Tecnologias e utilidade
- **Python 3.10** – Linguagem de programação utilizada para todo o desenvolvimento.
- **OpenCV** – Biblioteca para processamento de imagens e vídeo, usada para captura e análise da webcam.
- **MediaPipe** – Biblioteca para detecção e rastreamento facial, análise de olhos e landmarks.

## Funcionalidades
- Detecção facial
- Análise de olhos (Eye Aspect Ratio – EAR)
- Classificação de estado de atenção: **ATENTO** / **DISTRAÍDO**

## Requisitos
- Python 3.10
