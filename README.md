# Attention Detector
 
Sistema de visão computacional em **tempo real** para monitorar e detectar a atenção do usuário.

## Tecnologias e utilidade
- **Python 3.10** – Linguagem de programação principal.
- **OpenCV** – Processamento de imagem e vídeo.
- **MediaPipe** – Rastreamento facial e análise de olhos.

## Funcionalidades
- Detecção facial
- Análise de olhos (Eye Aspect Ratio – EAR)
- Classificação de estado de atenção: **ATENTO** / **DISTRAÍDO**

## Requisitos
- Python 3.10  
> Recomendado para compatibilidade com todas as dependências
